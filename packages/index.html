PopTools 扩展仓库

如何在Blender中添加此仓库

  1. 打开 Blender → 偏好设置 → 获取扩展 → 仓库
  2. 点击 "+" → 添加远程仓库
  3. 设置URL为: https://你的用户名.github.io/你的仓库名/index.json
  4. 启用"启动时检查更新"
